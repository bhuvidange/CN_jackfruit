# CN_jackfruit
CN jackfruit-traffic junctions simulation

Smart Traffic Signal Monitoring System:
A distributed monitoring system where traffic signal controllers report real-time traffic data to a central control unit.

Making use of SUMO (Simulation of Urban MObility) simulation software along with TraCI (Traffic Control Interface)
API to simulate various traffic scenarios likening to real life. 

Making use of Socket Programming and concepts of UDP packet transmission to create a fast congestion monitoring
system that informs us of the changing congestion rates throughout the lifespan of the simulation
